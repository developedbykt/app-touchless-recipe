利用規約

最終更新日：2026年6月14日

タッチレス・レシピ（以下「本アプリ」）をご利用いただく前に、以下の利用規約をお読みください。本アプリをご利用になることで、本規約に同意したものとみなします。

第1条（サービスの内容）

本アプリは、YouTube動画を音声操作および近接センサー操作で制御しながら料理レシピを管理するAndroidアプリケーションです。主な機能は以下のとおりです。

  • YouTube動画の登録・再生（通常動画およびYouTube Shorts対応）
  • 近接センサーによるハンズフリー操作
  • 音声コマンドによる再生制御
  • 料理タイマー機能（タブ内タイマーおよびレシピ再生中タイマー）
  • タグ・メモによるレシピ管理
  • 調理履歴の記録

本アプリはYouTube Data API v3を使用しており、YouTubeのサービス利用規約が適用されます。

第2条（禁止事項）

利用者は以下の行為を行ってはなりません。

  • 本アプリを違法な目的で使用すること
  • 本アプリのリバースエンジニアリング、改ざん、または複製を行うこと
  • 本アプリを通じて著作権を侵害するコンテンツにアクセスすること
  • その他、開発者が不適切と判断する行為

第3条（広告）

本アプリはGoogle AdMobによるバナー広告およびインタースティシャル広告を表示します。本アプリは無料で提供されており、広告収益によって運営されています。広告はGoogleの広告ポリシーに基づいて配信されます。

  • プレイヤー画面でのYouTube動画上への広告の重ね表示は行いません（YouTubeサービス利用規約の遵守）

第4条（端末機能の使用）

本アプリは以下の端末機能を使用します。

  • マイク：音声コマンド認識のため
  • 近接センサー：ハンズフリージェスチャー操作のため
  • バイブレーション：タイマー終了通知のため
  • 画面の常時点灯：料理中の画面消灯防止のため
  • 画面の明るさ調整：再生中の輝度固定のため

これらの機能は上記目的にのみ使用します。

第5条（知的財産権）

本アプリのコードおよびデザインに関する知的財産権は開発者に帰属します。本アプリを通じて表示されるYouTube動画の著作権は各権利者に帰属します。

第6条（免責事項）

本アプリは現状のまま提供されます。開発者は以下について一切の責任を負いません。

  • 本アプリの利用または利用不能により生じた損害
  • YouTube APIの仕様変更や停止による機能への影響
  • 端末やOSのアップデートによる動作への影響
  • 音声認識の誤認識による意図しない操作
  • 近接センサーの誤検知による意図しない操作
  • タイマー機能の誤動作による調理上の問題

第7条（データの削除）

本アプリには設定画面からレシピデータ・調理履歴・タグ・全設定をリセットする機能があります。一度削除したデータは復元できません。削除操作は利用者自身の判断と責任で行ってください。

第8条（サービスの変更・終了）

開発者は予告なく本アプリの機能変更、アップデート、またはサービスの終了を行う場合があります。

第9条（準拠法）

本規約は日本法に準拠し、解釈されるものとします。

プライバシーポリシー

最終更新日：2026年6月14日

Developed by kt（以下「開発者」）は、タッチレス・レシピ（以下「本アプリ」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

第1条（収集する情報）

【開発者が収集する情報】
本アプリの開発者は、ユーザーの個人情報を独自に収集・保存しません。すべてのレシピ・タグ・調理履歴・設定データはお使いの端末内にのみ保存されます。

【第三者サービスが収集する情報】
本アプリは以下の第三者サービスを利用しており、各社のプライバシーポリシーに従いデータが収集される場合があります。

第2条（YouTube Data API）

本アプリはYouTube動画（通常動画・YouTube Shorts）の情報（タイトル・サムネイル・概要）を取得するためにYouTube Data API v3を使用します。APIリクエスト時にYouTube（Google LLC）のサーバーと通信します。

  • YouTube利用規約
  • Googleプライバシーポリシー

第3条（Google AdMob）

本アプリはGoogle AdMobを使用してバナー広告・インタースティシャル広告を表示します。AdMobは広告配信の最適化のために以下の情報を収集する場合があります。

  • 広告識別子（広告ID）
  • デバイス情報（機種、OSバージョン等）
  • アプリの使用状況

Googleの広告に関するプライバシーポリシー

ユーザーはデバイスの設定から広告IDをリセットまたは広告のパーソナライズを無効にすることができます。

第4条（マイク・音声データ）

本アプリは音声コマンド機能のためにマイクへのアクセス権限を使用します。

  • 音声認識はAndroid端末のシステム機能（Speech Recognition API）を使用します
  • 日本語（ja_JP）および英語（en_US）の音声コマンドに対応しています
  • 録音された音声データは音声コマンドの解析にのみ使用されます
  • 音声データは開発者のサーバーへは送信されません
  • 音声認識処理はGoogleの音声認識サービスを経由する場合があります

第5条（近接センサー）

本アプリは手をかざすハンズフリー操作のために近接センサーを使用します。センサーデータは端末内での操作判定（動画再生/一時停止・タイマー操作）にのみ使用され、外部へは一切送信されません。

第6条（バイブレーション）

本アプリはタイマー終了時の通知のためにバイブレーション機能を使用します。バイブレーションはタイマー終了を知らせる目的にのみ使用され、外部との通信は一切行いません。

第7条（端末内に保存するデータ）

以下のデータはお使いの端末内にのみ保存されます。

【データベース（SQLite）に保存するデータ】

  • 登録したYouTube動画の情報（タイトル・サムネイルURL・概要・メモ・Shortsフラグ）
  • タグ情報
  • 調理履歴

【端末設定領域（SharedPreferences）に保存するデータ】

  • 近接センサー操作の設定
  • 音声認識の言語設定
  • 再生中の画面輝度設定
  • フォントサイズ設定
  • テーマ設定（ライト/ダーク/システム追従）
  • タイマープリセット
  • レシピごとのタイマー分数

これらのデータは開発者を含む第三者へ送信されることはありません。アプリをアンインストールするか、設定画面のデータ管理から削除することができます。

第8条（パーミッション）

本アプリが使用するAndroidパーミッションは以下のとおりです。

  • INTERNET：YouTube Data API・AdMobの通信に使用
  • RECORD_AUDIO：音声コマンドの認識に使用

第9条（子どものプライバシー）

本アプリは特定の年齢層を対象としておらず、13歳未満のユーザーから意図的に個人情報を収集することはありません。

第10条（プライバシーポリシーの変更）

本ポリシーは必要に応じて更新される場合があります。重要な変更がある場合はアプリのアップデートを通じてお知らせします。
